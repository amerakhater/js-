JavaScript setTimeout() Example

setTimeout() is used to run a function once after a delay (in milliseconds).

Example 1 – Simple Timeout

setTimeout(() => {
  console.log("This message appears after 2 seconds");
}, 2000);

Example 2 – With Parameters

function greet(name) {
  console.log("Hello, " + name);
}

setTimeout(greet, 3000, "Amira");

Example 3 – Cancel Timeout

let timeoutId = setTimeout(() => {
  console.log("This will NOT show");
}, 4000);

clearTimeout(timeoutId);
