Understanding this in JavaScript

The value of this depends on how a function is called.

Example 1 – Global Context

console.log(this);

Example 2 – Inside an Object Method

let person = {
  name: "Amira",
  greet: function() {
    console.log("Hi, I'm " + this.name);
  }
};
person.greet();

Example 3 – In a Regular Function

function showThis() {
  console.log(this);
}
showThis();

Example 4 – In an Arrow Function

let obj = {
  value: 100,
  arrow: () => console.log(this.value)
};
obj.arrow();

Example 5 – With Event Listener

document.querySelector("#btn").addEventListener("click", function() {
  console.log(this); // the button itself
});
